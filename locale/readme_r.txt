Руководство оператора
Назначение программы:
Программа предназначена для моделирования Овощной грядки.
Условия работы программы:
Для выполнения программы необходима, функционирующая под управлением операционной системы семейства Windows (не ниже Windows XP), ПВЭМ имеющая 64 битную разрядость системы.
Выполнение программы:
Для переключения языка пользователь должен нажать указаную в меню клавишу.
Выбор необходимого пункта меню осуществляется путем нажатия клавиш с числами (на каждой странице разное количество). Для возврата в предыдущее меню используется клавиша,описанная в данном экране меню.
