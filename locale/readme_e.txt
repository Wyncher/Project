Operator's manual
Purpose of the program:
The program is designed to simulate a vegetable garden.
Program working conditions:
To execute the program, it is necessary to operate under the operating system of the Windows family (not lower than Windows XP), a PVEM with a 64-bit system capacity.
Program execution:
To switch the language, the user must press the key indicated in the menu.
The selection of the required menu item is carried out by pressing the keys with numbers (there is a different number on each page). To return to the previous menu, use the key described in this menu screen.
